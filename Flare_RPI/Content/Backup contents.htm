Production Report

The RPI Production Report is an available feature of the software. Each time you stop a Virtual Profile, the software prompts you with the choice to view a Production Report with user selectable dates and range. See Figure 98

 

To use this capability, you must first manually enable the function by browsing to the C:\RPI\Log\KIC2000ReadOnly.kiccfg file, and change the following line to =1

[REPORT_MANAGEMENT]

UseReportManagement=0

 

The Production Report displays a DPMO (Defects Per Million Opportunities) chart and a Reflow Yield chart, along with the details for each respective chart.

 

Figure 98: Production Reports

Tips

  • The Production Report can be launched manually by locating and running the following file: C:\RPI\Report\index.htm.

 

  • The Production Report requires Active-X so make sure your internet browser is set to allow Active-X content.

 

 

Production Report Viewer

Production report settings – The Settings button displays a menu where you can enable or disable the view of each chart and set the limit values (red lines on graph -- See Figure 98). Click on the “Setting” button to open the settings dialog. See Figure 99

Figure 99: Production Report Settings

 

DPMO chart – Turns the display of the DPMO Chart On/Off.

 

Limit line – When DPMO Chart is enabled (On), sets the location of the DPMO Chart limit line (red line).

 

Yield chart – Turns the display of the Yield Chart On/Off.

 

Limit line – When Yield Chart is enabled (On), sets the location of the Yield Chart limit line (red line).

 

Save – Click to save the settings and redisplay the graph(s) using those new settings.

 

X – Close the settings dialog without applying any changes.

 

 

DPMO Chart

Defects Per Million Opportunities (DPMO) is a measure of process performance that defines a defect as a nonconformance of a quality characteristic to its specification. DPMO differs from defective Parts Per Million (PPM) in that it takes into account the possibility of multiple defects in one part. 

 

Figure 100: Production Reports – DPMO Chart

 

 

DPMO Chart Details

 

Figure 101: Production Reports – DPMO Detail

 

The Details for the DPMO Report are as follows (See Figure 101):

# Boards: The number of units produced

# TC Points: The number of thermocouple data points measured

# Profile Specs: The number of profile specification data points measured

Total # Opportunities: The number of opportunities for a defect for all units produced

Total # Errors: The number of defects (statistics that are out-of-spec) for all units produced

DPMO: Defects per million opportunities

Reflow Yield Chart

The Reflow Yield Chart measures the products defect rate. Reflow Yield is a board level type of defect measurement.  Five errors produced in any one board defines it as defective. The yield appears as a percentage.

 

Figure 104: Production Reports – Yield Chart

 

 

Reflow Yield Chart Details

Figure 105: Production Reports – Yield Details

 

The Details for the Reflow Yield Report are as follows (See Figure 105):

 

# Boards: The number of units produced

# Boards with error(s): The number of defective units.  Five out-of-spec statistics per unit = One defective unit.

Yield: The successful yield rate (non-defects) expressed as a percentage.
