Multi-Lane System

For ovens with (4) lanes, a multi-lane system is also available. This system is a hybrid of the Dual Lane/Single System and Dual Lane/Dual System configuration described in previous section. The operation is similar to the Dual Lane/Single System operation described above, but across two instances of software.

 

Two complete systems are installed on the single machine. Hardware includes:

  • 2 pairs of KIC probes – one single probe on each lane
  • 2 DAU’s
  • 1 Encoder
  • 4 Board sensors
