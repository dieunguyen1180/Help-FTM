Introducing the PRI System

Pyrometer Reflow Inspection (PRI) is the first ever reflow inspection system to utilize pyrometers installed throughout the oven process tunnel.  These pyrometers measure the real-time temperature of production boards directly, without making contact, sensing their infrared energy as they go through the oven. 

 

The Pyrometer Reflow Inspection system is a comprehensive suite of thermal process management tools designed to give you the ultimate in PCB soldering quality control. KIC PRI can monitor the entire thermal process, giving advanced warning before a product drifts out of specification.  Its barcode control features can prevent costly product/recipe mismatches, and its power management features enable efficient operation at the lowest energy cost.

 

PRI combines software and hardware components into an integrated system, supplementing its thermal profile analysis core with special tools for performance optimization, changed variable prediction, and data file management.

 

 

The Hardware

The major hardware components to a PRI system are:

  • Data Acquisition Unit (DAU)
  • Pyrometer sensors
  • Purge Blocks
  • Conveyor speed encoder
  • Product sensor
  • Alarm Relay/Light Tower
  • Profiler

DAU

The DAU is the main control unit for the system, to which all installed sensors connect through. It interfaces with the computer using an Ethernet cross-over cable that connects from the UpLink port on the DAU directly to a USB to Ethernet adapter (or to an installed Network Interface Card (NIC)).  Please follow manufacturers’ instructions to install the USB adapter or NIC. 

 

The computer used with the PRI system can be a machine that is dedicated to only supporting PRI or it can also serve as the oven controller PC.  PRI can interface directly with the oven controller on selected models from many oven manufacturers.  When installed on an oven that is capable of communication with the PRI software, it will allow information (zone setpoints, conveyor setpoints, etc.) to be exchanged between the PRI software and the oven controller software.

 

The DAU must be plugged into an AC power source.  The DAU utilizes a universal A/C transformer power supply.  The DAU ships from the factory with multiple AC connectors to accommodate any connector/voltage requirement.

Pyrometers

The pyrometers are installed in the oven process tunnel.  They are mounted on the fixed rail, or in a manner that allows clear visibility of the printed circuit boards without interfering with the product flow by oven manufacturer approved brackets. The pyrometers are connected to an adapter box, which adapts the inputs to the DAU.  Typically half the pyrometers enter from the oven entrance and half enter from the oven exit. Pyrometers 1-6 are connected into one adapter box that will be plugged into the A labeled input of the DAU and pyrometers 7-12 are connected into the other adapter box and plugged into the DAU input labeled B. 

Nitrogen or Compressed Air Purge Block

Dual Purge Blocks supply a regulated flow of gas or air to each of the pyrometer sensors.

Conveyor Speed Encoder

The conveyor speed encoder is a sensor mounted to sense the teeth of a gear that turns while the conveyor is moving.  The conveyor speed encoder connects to channel #1 of the DAU Sensor ports.

Product Sensor

The product sensor is typically mounted near the entrance of the oven, on the fixed conveyor rail.  The sensor is connected to channel #2 of the DAU Sensor ports.  The first pyrometer acts as the main board sensing device, but the product sensor at the entrance of the oven is used to display the boards as they travel from the oven entrance to the first pyrometer (middle of zone 1).

Alarm Relay/Light Tower

The Alarm Relay connects to the Alarm/Barcode port of the DAU and provides a method for external notification to the user when an alarm condition occurs. It allows for a KIC supplied audible light tower to be powered and triggered; it can include SMEMA interface connections; it also allows a user to hardwire in an external device of their choosing. The Alarm Relay is required to have an external AC power source.

There are two audible light towers available – one model connects directly to the Alarm Relay device; the other model connects directly to the USB port of the PC. See the Light Tower section.

 

Profiler

The profiler is a separately offered data acquisition component that is required to use the PRI system.  It provides the on-board, through-the-process temperature data, what we call the “Thermocouple Profile”.

Profiler Models

The following profiler models are compatible with the PRI system software:

  1. SPS Smart Profiler
  2. X5
  3. K2

The SPS profilers are available in two different configurations —datalogger and datalog/wireless.  In standard datalogger mode, the profiler records temperature data as it moves through the thermal process.  A download cable is then connected to download the data to the PC running the PRI software.  In wireless mode, the profiler transmits data by Wi-Fi via a Smart Dock connected to the PC.  The PRI software displays the accumulating data in real-time.

 

For specific information regarding your profiler, consult one of the following KIC publications:

  • SPS Smart Profiler Hardware Guide, (Publication Number PTG-330310-000)
  • X5 Profiler Hardware Guide, (Publication Number EXP-337310-000)
  • K2 Profiler Hardware Guide, (Publication Number EXP-331310-000)

 

 

Additional Components

Barcode Reader

A user-supplied barcode reader can connect to the DAU Alarm/Barcode port (when using a DB-9 serial connection), to one of the DAU USB ports (when using a USB barcode scanner), or to a PC Com or USB port.  When used in conjunction with the Alarm Relay, the serial barcode reader hardware can connect to the DAU via the Alarm Relay Y- cable.  The barcode reader is mounted on the entrance side of the oven, on the in-feed conveyor. For additional information about using barcode scanners, see Appendix C – Barcode Tab.

Install The Software

Minimum PC System Requirements

  • 3 GHz Dual Core processor
  • 4GB RAM
  • SSD is recommended
  • Video 1024 x 768 / 16-bit color
  • 1 available USB port for data download
  • 1 available USB port (for software protection dongle key)
  • 1 available Ethernet port or 1 available USB port with Ethernet-to-USB adapter

 

Note:  Additional powered USB ports may be needed for additional accessories.

Notes Before Installation

  • Shut down all other applications as a precaution against software conflicts. 
  • If you are installing the PRI software on an oven that is capable of communication with the software, DO NOT place the PRI icon in the Windows Startup folder.  If the PRI software loads before the oven controller software, it can cause the oven controller software to malfunction.

 

Install

  1. Insert the USB flash drive into a USB port on your computer.

 

  1. Depending upon your computer setup, a dialog box may appear. If it does, select Openfolder to view files. If a dialog box does not appear, use Windows Explorer to browse to the USB flash drive directory.

 

  1. Double-click the installation file in the root directory to begin the installation.

 

  1. Follow the instructions for the installation program as the screens appear.
    *When installing the software over an existing PRI directory, the installation will automatically create a backup folder of the existing directory. A warning message will be displayed when selecting an existing directory, and a checkbox can be de-selected if you do not wish to create a backup.
    The name of the backup directory created will be C:\ PRI _Old_MM-DD-YYYY

 

Note: If you have questions regarding your hardware or software configuration contact KIC Tech Support.

 

 

Start the Software

Note: The installation automatically adds a KIC folder to the main Windows Start Menu.

 

 

  1. Click on the Windows Start button to locate the PRI icon.

 

Note: The icon will appear in the Start button’s list of frequently used applications or you can click the All Programs command to find it in the KIC folder that appears among the displayed folder/icon list.

  

 

     

 

 

  1. Alternately, you can browse to the C:\PRI directory and double click the KICHost.exe file to start the application.

 

 

  1. A specifically programmed USB dongle key is required to run the PRI software. When starting, if you see the above message, insert the dongle key and click OK.
