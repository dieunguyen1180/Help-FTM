Process Window Setup

Figure 14: Process Window Setup

 

Process Window Name – Name of the Process Window file that includes the statistics chosen and limits for those statistics, along with whatever text appears in the Description field.

Solder Paste Menu – A ‘read only’ library list of numerous solder pastes along with the statistics and limits suggested by the paste manufacturer; also included is a user-defined option in the list which allows you to create a specification of your own.  (See below for additional details)

Change Specs Name – Opens an external application allowing you to customize the name of any individual statistic. (See below for additional details)

Edit Specs – Screen allowing you to edit or choose statistics and limits for a chosen solder paste or define your own specs.

Read Only Text Box – For the given Process Window chosen it states the paste name, statistic’s used, and their cooresponding limits.To edit select the Edit Specs button.

Same specs for all TCs – By deselecting this checkbox, you can assign separate specifications for each individual thermocouple you are using. This option would be used if you had component specific specifications that differ from the general solder paste specs. Another use for this would be if you wanted to monitor the actual board temperature as well as component temps.  You would then only select the statistics for that TC that are relevant. If you wish to use the same specifications for all thermocouples, put a check mark in the box.

Select TC to view – This dialog box will appear only if the Same Specs for all TCs checkbox is deselected.  By clicking on the dropdown menu, you can view the specifications that have been defined for that number thermocouple.  If a description was included, it appears next to Label.

Process Window Description – Field allowing for freehand notes for a particular Process Window.

 

Solder Paste Menu

 

Figure 15: Solder Paste Menu

 

Solder Paste Menu - Once you have clicked on the Solder Paste Menu you will have a list of pastes to choose from.  Use the scroll bar on the right to find your paste, and then click on the paste in the list. 

 

  • Clicking the paste jar and tube button will give you technical support information for the solder paste manufacturer that is currently highlighted in the list.

 

  • Clicking the checkmark button will accept that paste and load its reflow specifications.

 

  • Clicking the X will cancel your selection and return you to the Process Window screen without making any changes.

 

Edit Specs

 

Figure 16: Process Window Edit Specs

 

The name of the solder paste selected from the Solder Paste Menu appears at the top of the screen.

 

There is a single drop-down list at the top that contains all of the available specifications that can be applied to your Process Window.  These specifications are specific limits or a group of limits that define the overall Process Window for your product.  These specifications include Slope, Preheat, Soak, Peak temperatures, and Time above temperatures.

 

Spec Details – The Spec Details coincide with the selected Specification check boxed in blue.  For each Specification selected, enter the Temperature, and Time limits.  These limits are usually dictated by the solder paste used.  The limits are used to measure the profile PWI.

 

CAUTION: Changes made within this screen will have a direct effect on the profile PWI value.

 

Once you have completed all spec modifications you can click on the check (Save and Exit) button and your changes are applied.  Clicking on the X button will cancel your changes and exit.

 

 

Specify Different Specs for Individual TCs

 

Figure 17: Edit Specs screen with TC Selection & Label panel displayed

 

TC Selection & Label

If you have deselected the Same Specs for all TCs option on the Process Window Setup screen, when the Edit Specs screen appears, it will display an additional TC Selection & Label. 

 

  • Select – Use the Select drop-down menu to select the TC whose specifications you wish to view/edit.  The Spec Details for the selected TC appear.
  • Label – This area lets you type in a description or label to identify that particular TC. If left blank, the TCs will simply be identified as TC2, TC3, etc.

Note: When using separate specs, the Edit Specs screen is the only place where you can select or deselect which TCs are used for a profile.

 

Once you have completed all edits of the specs you can click on the check button and your changes are applied.  Clicking on the X (Cancel) button will cancel your changes and exit.

Change Specs Name

Figure 19: Save Process Window

 

If you wish to rename the label for a particular statistic, start by clicking on the Change Specs Name button, which opens a separate window. Select the check mark adjacent to ‘Use Change Specs Name’ at the top of the screen to enable this change function. Next, locate the desired specification in the greyed out columns, and type in the new name in the corresponding editable column. Click Save, then Exit to close the window and return to the Process Window screen.

 

Save - Process Window

Figure 20: Save Process Window

 

To create a Process Window, follow these steps:

 

  1. Choose a solder paste from the Solder Paste Menu.
  2. Edit the specs if necessary. 
  3. Name the Process Window
  4. Save the Process Window by clicking the check (save and exit) button

    A dialog box appears asking if you want to add this Process Window to your current list of Process Windows. 

 

  1. Click Yes will save it and exit to the main screen.

    You can click No and then click the X (cancel) button to exit without accepting or saving any changes.

Import Legacy Process Windows

You can import process windows that you have created in RPI or other KIC software into PRI and use them in your current production. 

 

  1. Open the RPI main product folder.

 

  
  1. Open the RPI Process Specs folder:
  
  1. Copy the files that you want to run in PRI.

 

  
  1. Open the PRI main product folder and then open the PRI Process Specs folder.
  

  1. Paste the copied files into the folder.

 

  

  1. Run the PRI software.


  1. On the PRI Main screen, click the Define Edit/Process Window button.

 

  
  1. The Process Window Name screen appears, showing the copied files available for selection in the drop-down menu.
