Hardware Status

 

Figure 22: Hardware Status Screen

 

Oven Controller – When connected to an oven that can communicate with the PRI software, the software will display oven status and display the oven control software name.

 

Profiler – Shows the status of the profiler.  This box displays the following information:

  • Profiler communication status
  • TC Temps– for all connected thermocouples
  • Battery – percentage of battery life
  • Internal Temperature – currrent temperature inside the profiler
  • Maximum Temperature Capability – the maximum temperature the profiler can read

 

Software key – Displays all the features currently programmed on the USB Dongle key.

 

Pyrometers – The current live temperatures for the pyrometers are displayed. If you do not see any data displayed in this field, the PRI software is not communicating properly with the DAU hardware. 

 

Note: If you are using the barcode capabilities, the last barcode read by a scanner appears.
