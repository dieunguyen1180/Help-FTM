Attach Thermocouples

Thermocouples (TCs), attached to solder joints or other important sites on the board, directly measure the temperature at the point that the TC bead is in contact with the PCB. The TC measurements are collected by the profiler and the KIC software to create thermal profiles.

 

Two methods are typically used to attach TCs—aluminum tape and high temperature solder. Both methods are discussed below.

 

The system works with multiple TCs—Air TC, Dwell TCs, and Product TCs – depending on what type of data the user want to collect. Product TCs record the temperature data for various sites on the board. The Air TC gets specific positioning because its temperature triggers the start of the profile data processing, aids with data alignment, and is used in the prediction capabilities of the software. Dwell TCs will be used for collecting contact time in the wave.

Attach The Air TC

Important considerations regarding the Air TC:

 

 

  • It must be attached at the leading edge of the board, extending approximately one inch (25 mm) in front of the leading edge of the board.

 

 

  • It must be plugged into channel 1 on the profiler.

 

  • If the user selected Wave On as the application type, ensure that the Air TC is angled downward so that it can make contact with the solder wave.

    Click the Forward Arrow button.

Attach Dwell TCs

If Dwell Time is selected, two TCs measuring Dwell Time will be required.

 

These TCs will need to be secured to the bottom left and right side of the board. Both TCs must contact the molten solder as the board is passing over the solder wave. The screen at the right shows one example of how these TCs might be attached. The user may also be able to wrap aluminum tape (or similar from the top side, around the front edge, to secure the TCs.

 

Note: The layout, or physical characteristics of the product being profiled will dictate how the user may be able to attach them, but for repeatable data, ensure the TCs are securely attached no matter which method is used.

Profile Retransmission

Note: For users of profilers with wireless capability.

 

While the profiler transmits the live profile data to the computer running the WPI software, it simultaneously stores the profile data in its onboard memory.

 

Once the KIC profiler detects that all of the TCs have cooled below the profile-end trigger value, it will begin retransmitting the profile data. The status of the profiler transmission will display in the lower left corner of the graph screen.

 

Figure 37: Profile download status

 

When using the Data logger version, the user must remove the KIC profiler from the machine and thermal shield and then connect the USB download cable to the KIC profiler. The profile data will automatically begin downloading to the WPI software.

CAUTION: The profiler and the product may be hot when exiting the machine. Use gloves when handling.

 

When all the data packets have been received, the software will display a message asking you to turn the profiler off, choose OK. Note: Failing to turn the profiler OFF will drain the battery when using a K2 profiler.

 

After the KIC profiler has completed its retransmission, click OK. The user will be brought to the Graph screen, with the profile and statistics displayed for the current profile run.

Note: If the Air thermocouple was more than 10C cooler than any other TC, the user will get an error message. The user must now check to see that the Air TC is plugged into

the first channel on the KIC profiler and that all the other TCs are firmly attached to the product. Then the profile will need to be rerun.

 

If the message, “Waiting for the board to exit” appears, wait till “Turn off the profiler” message appears and follow any subsequent messages that may appear.

 

Next, the software will automatically analyze the profile data and present the profile and statistics.

Optimization Tab

Figure 57: Profile Optimization Tab

 

The Optimization tab allows the user to set the search parameters for Navigator. The software will search through millions of setpoint and conveyor speed combinations to find the optimal machine settings for each product. These optimization calculations are based on the selections made on this tab.

 

Original PWI – The Process Window Index for the original profile

 

Best PWI – The best Process Window Index that can be found based on the machine recipe optimization constraints selected.

 

Speed change – The total change to the conveyor speed for the best machine recipe found.

 

Total setpoint change – The sum of all setpoint changes for the best machine recipe found.

 

Search Mode for Optimization

Minimize PWI – Search for the combination of setpoint temperatures and conveyor speed that will minimize the Process Window Index.

Allow Zone Setpoints To Change – This option will determine if Navigator will include zone setpoint changes when predicting new solutions. It also allows the user to exclude individual zones from suggested changes.

 

Allow Conveyor Speed To Change – Choose whether to allow Navigator to vary the conveyor speed. If this feature is chosen, the user can set the minimum and maximum speeds.

Maximize Conveyor Speed – Search for the setpoint temperatures that will maximize conveyor speed.

 

Minimize Energy Consumption – Used by the Power software feature. The software searches for the machine settings that will minimize the power consumption by finding setpoint solutions with slower conveyor speeds and lower temperature settings.

 

Conveyor Speed Constraints

As long as the Allow Conveyor Speed to Change feature is selected, these options are available.

  • Minimum – Input for the slowest conveyor speed Navigator can recommend for new products.
  • Maximum - Input for the fastest conveyor speed Navigator can recommend for new products.

 

Manual Profile Prediction

The WPI software has automatic (Navigator), and manual prediction capabilities. Manual prediction gives the user flexibility to easily predict changes to the machine settings (temperature settings, conveyor speed), and view the results without having to spend the time actually running unnecessary profiles. This feature is very helpful for minimizing the time spent fine-tuning or developing a thermal profile.

Make Changes

To predict changes to the machine settings, or modify the Navigator prediction results, click on the zone to change. In the example below, Zone 1 has been selected.

 

Figure 56: Prediction Settings

 

Type the new temperature setting, and then press Enter.

 

The Navigator prediction tool will predict the results of that change, and then automatically update the PWI, predicted Statistics, and the profile graph. The new predicted results appear on the graph in a dotted-line format. This format lets the user quickly determine the difference between the original and predicted profiles.

Note: When separate Top and Bottom setpoints are used, the Navigator prediction will maintain the same delta between the Top and Bottom setting that was used in the original profile.
