Profile Printing

TThe WPI software will print profiles only from within the profile display screens. Click on the Printer icon button at the bottom of the profile display to open the Profile printout setup menu. There are two print formats available – Portrait or Landscape.

Portrait Mode

Figure 163: Profile Print Format #1 Options

 

  1. Check the desired items wished to include on the profile printout.
  2. Press the Print button to print the report.
  3. Press the Print Preview Button to display a preview of the report.

Figure 164: Sample Portrait Print Preview.

 

Landscape Mode

Figure 165: Profile Print Portrait Options

 

Below are the additional options available with Landscape mode:

  • Memo Box – Enables/Disables a Memo Box area for Draft, Review and Approval Signatures and Dates.
  • Lead Free Logo – Enables/Disables the Lead-Free Logo display area.
  • Lead Free Logo Path – When “Lead Free Logo” is enabled, specify the path to any BMP image file that the user wants to appear in the Lead Free Logo display area on the report.

Figure 166: Sample Landscape Print Preview

 

Optionally, the user can print the contents of any screen in the WPI software by pressing F9. The F9 function will not work while viewing the Profile Printout Setup dialog box.

Note: Before printing, make sure to have defined a printer for use with Windows.
