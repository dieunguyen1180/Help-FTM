Hardware Status

 

Figure 22: Hardware Status Screen

 

Profile

Shows the status of the profiler. This box displays the following information:

  • Profiler communication status
  • Temperature display – for all connected thermocouples
  • Battery Voltage, Internal Temperature, Maximum Temperature Capability

Software Key

Displays all the features currently programmed on the USB Dongle key

Probes

This box displays data for:

  • Current live temperatures from the Odd and Even probes
  • The Board Sensor, Barcode Product Sensor, and Conveyor Speed Encoder.
    Note: If there is no data displayed in this field, the WPI software is not communicating properly with the DAU hardware.

Barcode

If the user has purchased and enabled Barcoding, this section will display:

  • Any Barcode features enabled
  • Last barcode read

 

Test Alarm Relay – Allows the user to activate/deactivate the external Alarm Relay device for testing or troubleshooting purposes. The Alarm Relay hardware must be properly connected, and the buttons appear only when the software is detecting the DAU hardware.

Note: The alarm relay is not energized until VP is first running and then goes out of spec, see Alarm Relay.
