Hardware Status

 

Figure 10: Hardware Status Screen

 

Profiler – Shows the status of the profiler. This box displays the following information:

  • Profiler communication status
  • Temperature display – for all connected thermocouples
  • Battery Voltage
  • Internal Temperature – Temperature inside the profiler
  • Maximum Temperature Capability – The maximum temperature the profiler can read
  •  

Software Key – Displays all the features currently programmed on the USB Dongle key.

 

Probes – The current live temperatures for the probes are displayed. A third line displays hardware information regarding the Board Sensor, Barcode Product Sensor, and Conveyor Speed Encoder. If you do not see any data displayed in this field, the WPI software is not communicating properly with the DAU/eTPU hardware.

Note: If you are using the barcode capabilities, an additional section will display showing the last barcode read.

 

Test Alarm Relay – Use this button to test the alarm relay function of the software. The alarm relay hardware must be properly connected. You can also use this button to manually disable the alarm, should it stay active after Virtual Profiling has been stopped. The button appears only when the software is detecting the DAU/eTPU hardware.

Note: The alarm relay is not energized until VP is first running and then goes out of spec. To learn more about this see the alarm relay section.
