Process Window Setup

 

Figure 13: Process Window Setup

 

Process Window Name – Name of the Process Window file that includes the statistics chosen and limits for those statistics, along with whatever text appears in the Description field.

Preheat Specifications

Edit Specs – Additional screen allowing the user to edit or choose statistics and limits for a chosen flux or define your own specs.

Flux Menu – A read only library list of fluxes along with the statistics and limits suggested by the flux mfg.; also included is a user-defined option in the list which allows to create a spec. of your own. See below for additional details.

Read only text box – Shows the Flux, Statistics Name, and Limits for the Preheat Process Window.

 

Wave Specifications

Read only text box – Shows the Statistics Name and Limits for the Wave Process Window.

Use Sweet Spot Target – When the Sweet Spot Target option is purchased and enabled it allows the user to manually define the target (0 % PWI) for each statistic. (See below for additional details)

Same Specs for all TCs – By deselecting this checkbox, the user can assign separate specifications for each individual thermocouple you are using. This option is useful for component specific specifications that differ from the general specs. Another usage senario is for monitoring actual board temperature as well as component temps. The user would then only select the statistics for that TC that are relevant. If the user wishes to use the same specifications for all thermocouples the box can stay selected.

Select TC to view – This dialog box will appear only if the Same Specs for all TCs checkbox is deselected. By clicking on the dropdown menu, the user can view the specifications that have been defined for that number thermocouple. If a description was included, it appears next to Label.

 

Process Window Description

Description box – Field allowing for freehand notes for a particular Process Window.

Change Specs Name – Opens an external application allowing the user to customize the name of any individual statistic. (See below for additional details)

 

Flux Menu

 

Figure 14: Flux Menu

 

When selecting the Flux Menu button, a pop-up containing a list of fluxes appears. Using the scroll bar on the right, the user can click their desired flux to select it. If the flux used is not on the list, the user will select the Define Your Own Spec option from the list, and then manually enter the process limits.

 

Flux jars – Displays a disclaimer information about the selected flux/process specifications.

 

Green check – Accepts the selected flux and loads its specs automatically. This then returns the user to the Process Window screen after being presented disclaimer information.

 

Red X – Cancels the user’s selection and returns them to the Process Window screen without making any changes.

 

 

Edit Specs

 

Figure 15: Process Window Edit Specs

 

 

This screen allows the user to choose and/or edit a variety of available statistics that will be calculated during profiling and production.

Spec Tabs

Preheat – When the user selects Preheat, it allows for modification of only the Preheat specs.

Wave – When the user selects Wave, it allows for modification of only the Wave specs.

Spec Selection Drop Down – This drop down allows for the selection of the spec the user would like to edit. The selected specs parameters will populate below.

Note: There will be additional, Wave specific, specifications not shown in the Preheat tab drop down menu. These specifications include Delta Between Preheat and Wave and Dwell Time

Spec Details

Spec Details Box – This box will populate all parameters that coincide within the selected Specification. The basic Preheat tab limits are created by the flux selected. Each tab has its own set of limits that are used to measure their own individual PWI.

CAUTION: Changes made within this screen will have a direct effect on the profile PWI value.

Green check – will apply any changes the user made.

Red X – will cancel any changes made and exit this screen.

TC Selection & Label

If the user has deselected the Same Specs for all TCs option on the Process Window Setup screen, when the Edit Specs screen appears, it will display an additional TC Selection & Label.

 

Select –The Select drop-down menu allows for selection of certain TC’s whose specifications the user wishes to view/edit. The Spec Details for the selected TC appear.

 

Label – This area lets the user type in a description or label to identify that particular TC. If left blank, the TCs will simply be identified as TC2, TC3, etc.

 

Note: When using separate specs, the Edit Specs screen is the only place where the user can select or deselect which TCs are used for a profile.

 

Green check – will apply any changes the user has made.

 

Red X – will cancel any changes made and exit this screen.

Use Sweet Spot Target (Optional)

 

Figure 18: Use Sweet Spot Target

 

Normally, when selecting a specification, only an upper and lower limit is defined, and the “target” value is automatically the center of the upper/lower limits. If the user has purchased the Sweet Spot Target option, enabling this feature allows the user to manually define their own Target value for each statistic. This function may be desired when there is a defined specification window, but the desired optimal value may not be the exact center of the limits.

 

 

Change Specs Name

Figure 19: Change Specs Name

 

Change Specs Name – Selecting this button opens up the ChangeSpecsName utility.

Use Change Specs Name – This check box allows the user to rename any label for a particular statistic. After locating the desired specification in the greyed out columns the user can type in the desired name in the corresponding editable white column. Some specifications require modifying more than one selection. Please contact your system supplier with any questions on using this utility.

Note: If the Use Change Specs Name selection box is unchecked it will still allow you to edit but any changes made will not be used in the WPI software..

Save – Selecting this will save any changes made.

Exit – Selecting this button will close the window and return to the Process Window screen.

 

Save - Process Window

Figure 20: Save Process Window

 

To create a Process Window, follow these steps:

 

  1. Choose a flux using the Flux Menu button.
  2. Edit Specs if necessary using the Edit Specs button.
  3. Type in the name using the Process Window Name text box.
  4. Save the Process Window by clicking the Green check button
    A dialog box will appear asking the user if they wish to add this Process Window to the current list of Process Windows.
  5. Clicking Yes will save it and exit to the main screen.

    Click No and then the Red X button will exit without accepting or saving any changes.
