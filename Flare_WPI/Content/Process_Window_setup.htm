Process Window Setup

 

Figure 11: Process Window Setup

 

Process Window Name – Name of the Process Window file that includes the statistics chosen and limits for those statistics, along with whatever text appears in the Description field.

Edit Specs – Screen allowing you to edit or choose statistics and limits for a chosen flux or define your own specs.

Flux Menu – A read only library list of fluxes along with the statistics and limits suggested by the flux mfg.; also included is a user-defined option in the list which allows you to create a spec. of your own. See below for additional details.

Read only text box – Shows the flux name, statistics name, and limits for a Process Window chosen, edited, or saved by you. The top half displays the specs chosen for the ‘Preheat’ side of the process, and the bottom half for the ‘Wave’ side of the process.

Same specs for all TCs – By deselecting this checkbox, you can assign separate specifications for each individual thermocouple you are using. This option would be used if you had component specific specifications that differ from the general specs. Another use for this would be if you wanted to monitor the actual board temperature as well as component temps. You would then only select the statistics for that TC that are relevant. If you wish to use the same specifications for all thermocouples, leave the box checked.

Select TC to view – This dialog box will appear only if the Same Specs for all TCs checkbox is deselected. By clicking on the dropdown menu, you can view the specifications that have been defined for that number thermocouple. If a description was included, it appears next to Label.

Process Window Description – Field allowing for freehand notes for a particular Process Window.

Change Specs Name – Opens an external application allowing you to customize the name of any individual statistic. (See below for additional details)

Use Sweet Spot Target - When the Sweet Spot Target option is purchased, enabling it allows you to manually define the target (0 % PWI) for each statistic. (See below for additional details)

 

Flux Menu

 

Figure 12: Flux Menu

 

Flux Menu - When you click the Flux Menu button, a list of fluxes appears. Use the scroll bar on the right to find your flux in the list and click to select it. If your flux is not on the list, choose the Define Your Own Spec option from the list, and then manually enter the process limits.

 

  • Clicking on the green check accepts the selected flux and loads its specs automatically. You then return to the Process Window screen after being presented disclaimer information.

 

  • Clicking the flux jars button displays a disclaimer information about the selected flux/process specifications.

 

  • Clicking on the red X cancels your selection and returns you to the Process Window screen without making any changes.

 

 

Edit Specs

 

Figure 13: Process Window Edit Specs

 

The Edit Specs screen allows you to choose and/or edit a variety of available statistics that will be calculated during profiling and production.

 

At the top left of the screen are two tabs – Preheat and Wave – allowing you to select and define different specifications and limits for the two different halves of the wave solder process. When you select a tab, it will display that name (Preheat in the example above) at the top of the screen to let you know which selection area you are working in.

There is a single drop-down list at the top that contains all available specifications that are available for that part of the Process Window. These specifications are specific limits or a group of limits that define the overall Process Window for your product. These specifications include Slope, Preheat, Peak temperatures, Dwell Time in Wave, etc.

 

Spec Details – The Spec Details coincide with the selected Specification. For each Specification selected, enter the Temperature, and Time limits. These limits are usually dictated by the flux being used. The limits are used to measure the profile PWI.

 

Caution: Changes made within this screen will have a direct effect on the profile PWI value.

Once you have completed all spec modifications you can click on the green check (Save and Exit) button and your changes are applied. Clicking on the red X button will cancel your changes and exit.

 

Specify Different Specs for Individual TCs

Figure 14: Edit Specs screen with TC Selection & Label panel displayed

TC Selection & Label

If you have deselected the Same Specs for all TCs option on the Process Window Setup screen, when the Edit Specs screen appears, it will display an additional TC Selection & Label.

 

  • Select – Use the Select drop-down menu to select the TC whose specifications you wish to view/edit. The Spec Details for the selected TC appear.
  • Label – This area lets you type in a description or label to identify that particular TC. If left blank, the TCs will simply be identified as TC2, TC3, etc.

 

Note: When using separate specs, the Edit Specs screen is the only place where you can select or deselect which TCs are used for a profile.

 

Once you have completed all edits of the specs you can click on the green check button and your changes are applied.  Clicking on the red X (Cancel) button will cancel your changes and exit.

Use Sweet Spot Target (Optional)

 

 

Normally, when selecting a specification, only an upper and lower limit is defined, and the “target” value is automatically the center of the upper/lower limits. If you have purchased the Sweet Spot Target option, enabling this feature allows you to manually define your own Target value for each statistic. This function may be desired when there is a defined specification window, but the desired optimal value may not be the exact center of the limits.

 

 

Change Specs Name

If you wish to rename the label for a particular statistic, start by clicking on the Change Specs Name button, which opens a separate window. Put a check mark in the checkbox at the top of the screen to enable the change function. Next, locate the desired specification in the greyed out columns, and type in the new name in the corresponding editable column. Click the Save, then Exit to close the window and return to the Process Window screen.

 

Save - Process Window

Figure 8: Save Process Window

 

To create a Process Window, follow these steps:

  • Choose a flux from the Flux Menu.
  • Edit the specs if necessary. 
  • Name the Process Window
  • Save the Process Window by clicking the green check (save and exit) button.

    A dialog box appears asking if you want to add this Process Window to your current list of Process Windows.

  • Click Yes will save it and exit to the main screen.

    You can click No and then click the red X (cancel) button to exit without accepting or saving any changes.
